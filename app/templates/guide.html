Para fazer requisições, use a caixa de busca da página da api. Nela, você poderá fazer os quatros tipos de requisições: GET, POST, PUT e DELETE.

Rotas para fazer as requisições:

  •  Get: usuarios
  •  Post: create_user
  •  Put e Delete: não precisa de rota, basta preencher os campos (no caso do put) e selecionar o usuário (é preciso fazer um get antes de tentar fazer put ou delete, para se obter os dados)

Python

from json import dumps
import requests 

url = f"https://niceeapi.herokuapp.com/api/v1/"

def get_data(): 
    return requests.get(url+"/usuarios").json()

print(get_data()) 
            
url = f"https://niceeapi.herokuapp.com/api/v1/"
def post_data():
    data = {
        "name":"test",
        "email":"test@gmail.com",
        "password":"test123",
    }
    res = requests.post(url+"/create_user", data=dumps(data))
    print(res)
    return res

post = post_data()
print(post) # Response [200]
